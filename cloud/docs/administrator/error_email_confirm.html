Ошибка Ссылка не действительна

Ссылка из письма-приглашения не действительна. Так может быть по нескольким причинам:

  • Прошло слишком много времени с момента отправки приглашения. Ссылка устарела.
  • Ссылка могла плохо скопироваться из письма в адресную строку браузера. Попробуйте еще раз перейти по ссылке.
  • Администратор Си-Норда мог по ошибке отправить вам письмо-приглашение и затем его удалить.

Удачи!
