Пульт с указанным UID «привязан» к другому партнёру.

Пульт может быть привязан только к одному партнёру. Для того, чтобы привязать указанный пульт к вашему аккаунту, необходимо связаться с этим партнёром и попросить его «отвязать» пульт от своего аккаунта.

Для этого нужно в Центре охраны сгенерировать новый UID: Менеджер событий / Связь с облаком / UID Центра охраны / Сгенерировать новый UID.

UID можно узнать в Центре охраны, в меню «Связь с блаком».
