Подключение роутера


Центр охраны с указанным UID-ом не зарегистрирована в облаке.

Как узнать UID Центра охраны?

  1. Правый клик по Менеджеру событий;
  2. из выпадающего списка выберите Связь с облаком;
  3. перейдите во вкладку UID Центра охраны.

Обновление ПО роутера


Инструкция по обновлению:

  1. Скачайте файл с расширением 7z: http://www.cnord.ru/download/box/
  2. Загрузите файл, используя форму ниже.
  3. Перезагрузите роутер.

Перезагрузка роутера


Внимание:

  1. По нажатию на «Перезагрузить роутер», фактическая перезагрузка произойдет не сразу, а в течении 5 минут.
  2. Перезагрузка роутера занимает несколько минут — в это время колл-центр работать не будет.
  3. Пока роутер перезагружается, звонки на номер колл-центра будут переводиться на оператора.
